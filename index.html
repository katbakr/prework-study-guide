HTML

  • HTML stands for Hypertext MArkup Language. Used to give content structure.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Head element is a container for metadata.
  • Meta elements contain information about the page that is used by the browser.
  • Child elements are elements inside of a parent element.
  • HTML often represented by angle brackets.
  • Forward slashes used to create closing tags.
  • H1-H6 elements used to show different levels of heading to create titles or subtitles.
  • Use img element to add information for images shown on the website
  • ul, ol, and li elements are used to make unordered lists.
  • p element used to enclose paragraphs or lines of text. Used often to add text to webpages.
  • br element is used to create a line break between two blocks of content.
  • Footer appears at the bottom of a page and contains author, copyright, etc.

CSS

  • CSS stands for cascading style sheets. Gives conent style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • If I want to style an element directly in the HTML file, use inline CSS.
  • Internal style contains CSS information for the webpage in the head section of the HTML file. This is used to identify elements,and to style multiple elements in the code.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations give two pieces of information about the code: the CSS property applied and the value of the property.
  • Margin property indicates how much space ther is outside of the element on the webpage.
  • Use CSS box model to better understand style organization.
  • Can use hexadecimal code to assign new colors to designs.
  • Write DRY code to minimize repetion and extra work.
  • rel attribute defines the relationship between current doc and linked source.
  • href attribute defines location of the external resource.
  • Use style element to in the head element to add stylesheets directly to an HTML file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • Commit regularly.
  • Feature branches are used for CSS and JS in our example.
  • Use git add -A to stage changes to be commited
  • Use git commit -m and add a message about changes made to the application
  • Use git pull origin main to sync local and remote branch in GitHub.
  • Use git push origin feature/your-project.
  • Last complete pull request through GitHub. (reference gitflow on prework to see steps.)
  • After, use git checkout main and git pull origin main to update local repo.

JavaScript

  • JavaScript is used to add interactivity to the website.
  • It determines the response to data entry forms.
  • A semicolon is used at the end of a line to indicate where the statement ends.
  • JavaScript is case sensitive.
  • A string is a sequence of text.
  • Boolean is true/false statements.
  • An array allows you to store multiple values in a single reference.
  • You can add comments to explain a line of code and it will be ignored by the browser.
  • Conditionals are used to test if an expressions is tue or not. Use if...else statements.
  • Functions are a way of adding functionality that can be reused.
  • Can only use a function when it is called by name in code.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
